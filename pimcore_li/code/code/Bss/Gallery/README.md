#Bss Gallery for Magento 2

```

step 1 : upload folder in folder_dir to web/root folder (ex : public_html)
step 2 : upload other folder to public_html/app/code/Bss/Gallery folder
step 3 : active module : bin/magento setup:upgrade
step 4 : generation static files : bin/magento setup:static-content:deploy
step 5 : clear cache : bin/magento cache:flush
DONE!

```
